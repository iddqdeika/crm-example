# Implementation Plan: Ad Groups and Creatives on Campaign Creation

**Branch**: `010-creation-ad-groups` | **Date**: 2026-03-02 | **Spec**: [spec.md](./spec.md)  
**Input**: Feature specification from `specs/010-creation-ad-groups/spec.md`

## Summary

Enable users to add ad groups and creatives during campaign creation — in a single form submission — by extending the `POST /api/campaigns` backend endpoint to accept nested `ad_groups`, updating the frontend creation page to include the same collapsible accordion ad-group section as the edit page (via an extracted shared component), and verifying the change with backend integration tests, frontend component tests, and a new end-to-end Playwright scenario.

No database migrations are required. All existing behavior is preserved through backward-compatible additions.

## Phase 1: Design

### Backend Changes

#### `backend/src/schemas/campaign.py`

Add one field to `CampaignCreate`:

```python
class CampaignCreate(BaseModel):
    name: str = Field(..., min_length=1, max_length=255)
    budget: Decimal = Field(..., ge=0)
    status: CampaignStatus = CampaignStatus.active
    ad_groups: list[AdGroupUpsert] | None = None  # ← NEW
```

Validation: Any `AdGroupUpsert` item with a non-null `id` field raises 422 (id not expected on creation).

#### `backend/src/api/campaign.py` — `post_campaign`

After `campaign = await create_campaign(...)`, run the ad-group creation loop if `body.ad_groups` is not None:

```python
@router.post("", response_model=CampaignResponse, status_code=status.HTTP_201_CREATED)
async def post_campaign(body: CampaignCreate, ...):
    # Reject ad group items that supply an id
    if body.ad_groups:
        for item in body.ad_groups:
            if item.id is not None:
                raise HTTPException(status_code=422, detail="ad_groups items must not include id on creation")

    campaign = await create_campaign(db, current_user.id, body.name, body.budget, body.status)

    if body.ad_groups:
        for item in body.ad_groups:
            ag = await create_ad_group(db, campaign.id, ...)
            if item.creatives:
                for cr in item.creatives:
                    await create_creative(db, ag.id, ...)

    ag_with_cr = await _load_ad_groups_with_creatives(db, campaign.id)
    return _to_response(campaign, ag_with_cr)
```
