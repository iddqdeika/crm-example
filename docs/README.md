# Qualityboard

Adtech-themed landing, auth, and profile application.

## Quick start

See [Quickstart](../specs/001-adtech-auth-profile/quickstart.md) for:

- Environment setup (`.env.example` → `.env`)
- Running with Docker: `docker compose -f docker/docker-compose.dev.yml up --build`
- Local development (backend, frontend, migrations)
- Running tests (backend: `pytest`; frontend: `npm test`)
